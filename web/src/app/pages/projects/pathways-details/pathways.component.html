SCALE Pathways - Technical Advisor and Mentor

Jan. 2020 - Present

www.scalelearning.tech

Overview

SCALE Pathways is a web application for prospective students to explore universities and build sample schedules using real course data and dependencies. This project was originally created by me and two other developers as part of our senior design project for Washington State University. Since completing the prototype and graduating in December 2020, I have stayed on with SCALE to act as a technical mentor for future senior design teams as they continue to build upon Pathways. Most recently I have played a major role in getting the current version of the application deployed and publically available on the web.

Roles and Responsibilities

  • Developed prototype web application as part of my senior design project for Washington State University.
  • Technical mentor for 3 consecutive senior design teams from Washington State University.
  • Collaboration with multiple team members through bi-weekly sprint meetings and code review.
  • Responsible for maintaining the deployment infrastructure of the application and the integrity of the ongoing codebase.

Recent Contributions

  • Refactored API structure to generalize CRUD database operations and greatly reduce redundant code.
  • Added automatically generated Swagger documentation to API routes.
  • Created full deployment architecture on AWS using the AWS CDK.
  • Implemented a CI/CD pipeline within the deployment structure to automatically build and deploy the application in response to code changes.
